# PopoCard - Next.js製カードバトルゲーム

## 🎮 ゲーム概要
PopoCardは、戦略的な思考とリソース管理が要求される本格的なカードバトルゲームです。3種類の個性的なカードを組み合わせながら、4体の強力なモンスターに挑戦します。プレイヤーは限られたリソースを効率的に活用し、モンスターの攻略を目指します。

### 🌟 特徴
- 3種類のカード（赤・黒・青）による多彩な戦略性
- 連鎖的な効果を活用した奥深いゲームプレイ
- モンスターごとに変化する難易度
- ターン制による思考的なプレイ展開
- アニメーションによる視覚的なフィードバック

## 🎲 ゲームシステム

### カードの種類と効果

#### 🔴 赤のカード（攻撃カード）
- 基本的な攻撃手段として機能
- カードの数値分のダメージをモンスターに与える
- **重要な制限**: 直前に出した赤カードよりも大きな数値でないと使用不可
- 戦略的なカードの使用順序が重要

#### ⚫ 黒のカード（リソース消費カード）
- 山札からカードを捨て場に移動させる効果
- 捨てる枚数 = 14 - カードの数値
- リソース管理とリスク管理が重要
- 青カードとの組み合わせで効果が変化

#### 🔵 青のカード（特殊効果カード）
詳細な効果一覧：
1. **青1**: 手札の黒カードを強制的に使用（黒カードが必要）
2. **青2**: カードドローをスキップ（手札が減少）
3. **青3**: 次の黒カードの効果を3倍に増幅
4. **青4**: モンスターのHPを10回復（敵を回復させるリスクあり）
5. **青5**: 次の赤カードの攻撃力を半減（切り上げ）
6. **青6**: 効果なし（戦略的な使用が必要）
7. **青7**: 次の赤カードに+7の攻撃力ボーナス
8. **青8**: 捨て札から2枚をランダムに山札へ
9. **青9**: 捨て札から2枚をランダムに手札へ
10. **青10**: 場の赤・黒カードの枚数分ダメージ
11. **青11**: 次の黒カードの効果を半減
12. **青12**: 次の黒カードの効果を無効化
13. **青13**: 捨て札の枚数分のダメージを与える

### 🐉 モンスター
各モンスターは固有のHPと特徴を持ちます：
1. **GOBLIN** (HP: 30)
   - 最初の敵として基本的な難易度
   - 攻略のコツ: 基本的なカードの使い方を学ぶ
2. **DEVIL** (HP: 45)
   - 中級レベルの敵
   - 攻略のコツ: 効率的なダメージ効果の活用
3. **DARK_KNIGHT** (HP: 60)
   - 上級レベルの敵
   - 攻略のコツ: リソース管理とコンボの活用
4. **DARK_QUEEN** (HP: 75)
   - 最終ボス
   - 攻略のコツ: 全ての戦略を組み合わせた総合力が必要

### 📊 スコアシステム
現在の実装：
- モンスター撃破ボーナス（モンスターのHP値に応じて変動）
- ターン数ボーナス（少ないターン数でのクリアで高得点）
- 残り山札ボーナス

## 🎮 プレイ方法

### オンラインプレイ
1. [https://poposuke18.github.io/popoCard](https://poposuke18.github.io/popoCard) にアクセス
2. 「Start Game」をクリックしてゲーム開始

### ローカル環境での実行
```bash
# リポジトリのクローン
git clone https://github.com/poposuke18/popoCard.git

# プロジェクトディレクトリへ移動
cd popoCard

# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# ビルド
npm run build

# 本番環境での実行
npm run start
```
